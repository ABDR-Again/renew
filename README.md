# RENEW Landing Page

A highly optimized landing page for RENEW Pain & Wellness Center. Built with Vite.

## Setup
1. `npm install`
2. `npm run dev` to start the development server.
3. `npm run optimize` to compress images in `assets/images`.
4. `npm run build` to create a production build in `dist/`.

## Callback form

The callback form currently posts to FormSubmit using
`ru208208@gmail.com` as the temporary testing recipient. Replace that email in
the form action with `care@renewpainwellness.com` before launch. FormSubmit
requires each new primary recipient to activate the form from the confirmation
email generated by its first submission.
